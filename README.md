## introduction

edenfetch is a minimal fetch program written in rust. inspired by [jfetch](https://github.com/jimmysit0/jfetch).

## installation

```bash
git clone https://github.com/edenqwq/edenfetch
cd edenfetch/
cargo install --path .
```
